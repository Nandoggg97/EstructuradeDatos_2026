## Descripcion

Este proyecto corresponde al Entregable 3 de la materia de Estructura de Datos.

El programa implementa un simulador de arreglos dinámicos en C#, permitiendo crear un arreglo en tiempo de ejecución, capturar sus elementos desde consola, modificar su contenido y visualizar los cambios realizados. Además, incluye un reto adicional para demostrar la diferencia entre modificar el contenido de un objeto y reasignar una referencia en memoria (Heap).

## Tecnologias utilizadas

- C#
- .NET
- Visual Studio Code
- Git
- GitHub

## Compilar el proyecto

Abrir una terminal dentro de la carpeta del proyecto y ejecutar:

```bash
dotnet run
```

## Funcionalidades

- Crear un arreglo dinámico.
- Capturar elementos desde teclado.
- Mostrar el arreglo original.
- Modificar los elementos del arreglo.
- Mostrar el arreglo modificado.
- Demostrar el comportamiento de los tipos por referencia.
- Comparar la modificación del contenido contra la reasignación de una referencia.
